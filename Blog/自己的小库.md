### 在`js`中我们获取元素的`id`通常这样写

````js
var oId = document.getElementById('id')
````

### 我们如果这样写那下面我们会`var`出许多类似的，所以我的方法是这样

````js
function G(sID){
		return document.getElementById(sID);
}
G('id')
````
> 这样方便了许多不是吗
