<h2 style="font-weight: bold;">箭头函数【Arrow function】</h2> 

#### 概述

###### ES6的箭头函数的特点：永远是匿名函数，并且增加了ES5所没有的一些优点


###### //ES5

````js

var tt = function tt(){
	return 1 + 1;
}
````

###### //ES6

````js
var tt = () => 1+1  //是不是一对比，写法的差异就看出来了
````
